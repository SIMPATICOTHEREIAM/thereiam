File & Folder Guide

Everything lives in one folder on your computer. Here's exactly where to put new files so the site picks them up automatically.

THEREIAM (Startup)/
  img/  ← hoodie photos (.jpg or .png)
    hoodie-001IMG_1395.jpg
    ... (name them anything, then update the path in Hoodies or Brand tabs)

  Brand Photos/  ← optional folder for brand-only photos
    brand-photo-001.jpg
    ... (create this folder if needed)

  Music/  ← drop .mp3 or .m4a files here
    plain-sight.mp3
    ... (then add them in the Music tab above)

  logos/  ← logo files (don't rename these)
    THEREIAM LOGO1.png (wordmark)
    THEREIAM LOGO2.png (Mr. Smile)

  admin.html ← you are here
  thereiam-content.js ← master content file
  landing-preview.html
  page-hoodies.html
  page-brand.html
  page-archive.html

To swap logos: replace the files in logos/ keeping the exact same filenames. The site picks them up automatically.
